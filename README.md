# 公告管理網頁

光與夜之戀（Light and Night）遊戲內公告的編輯／管理工具。單頁應用，純前端，可直接放上 GitHub Pages 或本機雙擊開啟。

## 檔案結構（三個檔案要放在同一目錄）

| 檔案 | 用途 | 是否需讀取 |
| --- | --- | --- |
| `index.html` | 主程式（UI + 全部邏輯） | 改功能時讀這個 |
| `archive.js` | 內建歷史公告資料，`window.ARCHIVE = {months:[...]}`（約 793KB，672 則） | **大檔，不要讀進對話** |
| `xlsx.full.min.js` | SheetJS 0.18.5，Excel 讀寫用（已在地化，離線可用，約 864KB） | **大檔，不要讀進對話** |

> 維護提醒：改功能只需動 `index.html`，且盡量用小範圍修改（str_replace），不要把上面兩個大檔印出來或整檔重貼，否則 token 會飆高。

## 公告原始碼格式（與遊戲內一致，務必沿用此解析邏輯）

- `#|#`：分段分隔符。開頭為 `http(s)://` 的段落 → 圖片 banner；其餘 → 文字段落。
- `[rrggbb]文字[-]`：指定顏色文字，`[-]` 收尾。
- 換行：直接斷行（`\n` → `<br>`）。
- 標準配色（已做成一鍵按鈕）：內文 `6f6e7c`、標題 `b59974`（如 `┃開放時間┃`）、重點 `5c7bbb`（道具/角色名）、警語 `c04752`（※）。

解析時先 HTML escape，再把 `[hex]` 轉開 span、`[-]` 轉關 span。預覽面板刻意做成遊戲內米白樣式（米白底、深色字、上方「活動公告／系統公告」分頁、標題置中 + 「LIGHT AND NIGHT」裝飾字、banner 圓角）。

## 資料模型

- 工作資料 `DATA = [{month, hidden, items:[{...欄位}]}]`，存在 `localStorage`（key `gh_ann_data_v1`）。
- 首次載入由 `archive.js` 的內建資料種入；「還原內建」可捨棄編輯重設。
- 每則公告欄位：`發布日期、結束日期、標題、頁籤、內文、圖素、連結、ＩＤ、順位、已上正式服(bool)、測試服公告(bool)`。
- 「內文」= 公告原始碼。若「內文」為空但「圖素」有網址，預覽改用圖素當純圖片公告。
- 樣板存在 `localStorage`（key `gh_ann_templates_v1`）。

## 功能規格（目前狀態）

### 導覽列
- 分頁：歷史公告／編輯公告／樣板。
- 「＋ 開新公告」：跳到編輯頁開一則空白公告。
- 「匯入 Excel」「匯出 Excel」。
- 右側顯示儲存狀態（已儲存／提示訊息）。

### 歷史公告
- 左側為主區（清單，彈性寬，最小 380px）；右側預覽為固定欄（540px，遊戲內樣式）。
- 篩選：關鍵字搜尋（標題＋內文）、月份下拉。
- **可排序標題列**（Excel 式）：可點「公告／月份／開始時間／結束時間」排序，點一下切換升降序並顯示箭頭（▼新→舊／▲舊→新）；日期與月份欄預設降序、文字欄預設升序。**預設＝開始時間 ▼（新到舊，由上往下）**。左上角即時顯示「N 則」。
- 每張卡片：左為標題＋月份/頁籤標籤＋狀態圓點（綠＝已上正式服、金＝測試服公告），中間有「編輯／刪除」按鈕（滑過變清楚），右為開始時間與結束時間兩行。
  - 編輯 → 載入到編輯頁修改。
  - 刪除 → 確認後從該月份移除並存檔。
- 點卡片本體 → 右側預覽該則（標題 + 米白面板）；下方顯示月份/頁籤/發布/結束/連結/ID。
- 「＋ 在此月份新增公告」（需先選月份）、「還原內建」。

### 編輯公告
- 模式列：顯示「新公告」或「編輯：月份 / 標題」；儲存鈕文字隨之變「新增到此月份／儲存變更」。
- 「空白」開新、「套用樣板…」下拉、「存為樣板」、「全部複製」、「儲存」。
- 欄位：標題、所屬月份（下拉，可「＋ 新增月份…」）、頁籤、連結、發布日期、結束日期、ID、順位、圖素、已上正式服、測試服公告。
- 工具列：四色一鍵套用（選取文字包色 `[hex]…[-]`）、自訂色、`┃標題框┃`、插入圖片（自動補 `#|#`）、插入分段 `#|#`。
- 右側即時預覽（遊戲內樣式）+ 字數/段數/圖片數統計 + 配色圖例。
- 自動暫存草稿到 localStorage。

### 樣板
- 用目前編輯內容建立樣板、命名、設為預設（★，新公告下拉排最前）、重新命名、刪除。
- 匯出全部／匯入（合併或取代）成 JSON，供備份或團隊共用。
- 左側清單、右側樣板預覽（米白樣式）。

### Excel 匯入／匯出
- 格式：直向轉置 —— 每個工作表 = 一個月份（表名＝月份代號如 `2608`），A 欄為欄位名，每一直行為一則公告。
- 匯出：把目前 DATA 全部月份輸出成 `.xlsx`（布林維持 TRUE/FALSE）。
- 匯入：讀同格式 Excel，**取代**目前工作資料（樣板不受影響；匯入前會提醒備份）。已驗證 672 則往返後數量、標題、內文、布林完全一致。

## 已知限制／注意

- 資料與樣板存在本機瀏覽器，**不跨裝置同步**；團隊共用靠匯出/匯入 Excel（資料）與樣板 JSON。
- 圖片來源為 `backend.tw.gamehours.com`，未連 VPN 時預覽會顯示「圖片載入失敗」提示，屬正常。
- `archive.js` 為某次 Excel 的快照；要更新歷史，重新產生 `archive.js` 或直接用「匯入 Excel」。
- 時間目前照 Excel 原始值顯示（可能帶到秒）。
